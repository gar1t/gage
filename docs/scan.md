---
title: Scan
---

A _scan_ (or scan _run_) is the process of scanning sessions.

To start a scan, run:

```shell
gage scan
```

You're prompted to select _scanners_ for the run, along with what you'd
like to scan. You have a change to review the scan before starting it.

Gage runs each scanner over each of the selected sessions to find
issues.

## List scans

Gage maintains a record of each scan. To list scans, run:

```shell
gage scan list
```
