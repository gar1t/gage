---
title: Sessions
---

A _session_ is a file generated by Claude for each session. Sessions are
located under `~/.claude/projects/`.

List Claude sessions:

```shell
gage session list
```

Gage scans sessions by running small programs (called _scanners_) that look for
specific issues and provide help for resolving them.

You can view any Claude sessio file by running:

```shell
gage vession view [<SESSION_ID>]
```

If you omit the ID, Gage lets you pick from a list.

Gage's session viewer shows you low level data for each session entry. This is
what scanners see when they look for issues.
